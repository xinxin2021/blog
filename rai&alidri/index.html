然后请前往链接或点击蓝奏云下载地址(密码i4in)下载webdav-2.4.1.jar(如果您打开了上方的蓝奏云链接，可以在链接内直接下载)。

请新建一个文件夹，在里面放入.jar文件，随后创建一个.bat类型的文件，并使用记事本打开写入以下代码：
java -jar webdav-2.4.1.jar --aliyundrive.refresh-token="*************************"
